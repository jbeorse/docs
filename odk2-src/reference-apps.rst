Reference Applications
=============================

.. _reference-apps-intro:

The reference applications are presented here as real world examples of Data Management Applications built on the ODK 2 platform. The following documentation provides a guided tour of these application's workflow and modules, as well as an overview of the internal structure of its source files. The intention is to provide a useful example for organization's to use when creating their own Data Management Applications.
